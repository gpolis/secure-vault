## How to Use

### Encrypting a Password

1. Select "Encrypt" mode
2. Enter your password
3. Enter an encryption key (remember this!)
4. Click "Encrypt Password"
5. Copy the encrypted text and save it somewhere (database, notes, etc.)

### Decrypting a Password

1. Select "Decrypt" mode
2. Paste your encrypted text
3. Enter your encryption key (the same one you used to encrypt)
4. Click "Decrypt Password"
5. Copy the decrypted password
